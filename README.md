# Scripts Documentation

This directory contains a collection of useful shell scripts I use for system management and productivity. Each script is designed to perform a specific function, such as adjusting system settings, managing notes, or interacting with the clipboard.

## Available Scripts

- **battery-warner.sh**: Warns when battery is low.
- **brightness.sh**: Adjusts screen brightness.
- **cliphist.sh**: Manages clipboard history.
- **kblayout.sh**: Switches keyboard layouts.
- **notes.sh**: Simple note-taking utility.
- **power-menu.sh**: Provides a power management menu.
- **volume.sh**: Controls system volume.
- **wal.sh**: Changes wallpaper and applies color schemes.
- **wikiman.sh**: Searches and displays man pages from the web.
